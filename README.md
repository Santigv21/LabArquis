# SGV (Sistema de Gestión de Vehículos)

El Sistema de Gestión de Vehículos (SGV) es una aplicación diseñada para facilitar la gestión y administración de vehículos de manera eficiente y organizada. El objetivo principal del SGV es proporcionar una plataforma donde los usuarios puedan:

- Registrar vehículos y sus detalles.
- Realizar un seguimiento del mantenimiento y reparaciones.
- Gestionar citas y agendar servicios.
- Almacenar información relevante sobre los propietarios de los vehículos.
- Generar reportes relacionados con el uso y estado de los vehículos.

## Funcionalidades Principales

- **Registro de Vehículos**: Permite añadir nuevos vehículos al sistema con información detallada.
- **Gestión de Mantenimiento**: Registro de Historias de Mantenimiento y posibilidades de notificaciones para los usuarios.
- **Citas para Servicios**: Facilita la programación de servicios y reparaciones en talleres autorizados.
- **Reportes**: Opción de generar diversos reportes útiles para la gestión de la flota vehicular.

## Tecnologías Usadas

- [Node.js](https://nodejs.org/) como backend.
- [React](https://reactjs.org/) para el frontend.
- Base de datos [MongoDB](https://www.mongodb.com/) para el almacenamiento de datos.

## Instalación

1. Clone el repositorio.
2. Ejecute `npm install` en el directorio del servidor.
3. Inicie el servidor con `npm start`.
